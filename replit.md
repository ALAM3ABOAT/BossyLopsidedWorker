## Project Structure
- `app.py` - تطبيق Flask الرئيسي
- `templates/` - قوالب HTML
  - `index.html` - الصفحة الرئيسية
  - `login.html` - صفحة تسجيل الدخول
  - `admin.html` - لوحة الإدارة

## Features
- عرض التطبيقات بتصميم عصري
- البحث في التطبيقات
- تصنيفات (ألعاب، أدوات، تواصل، وسائط)
- لوحة إدارة لإضافة/تعديل/حذف التطبيقات
- نظام تسجيل دخول آمن

## Database
